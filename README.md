Free.RateLimit
==============

速率限制用于控制网络接口控制器发送或接收的流量速率，并用于防止DoS攻击。

大多数API受每秒或分钟，或另一个短时间段可以进行多少次调用的限制，以防止服务器超载并保持对其他客户端的高质量服务。
